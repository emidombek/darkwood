How to Play

In this game there are 8 cards, each card has a match that you as the player must find. All of matches must be found before the timer runs out in order to progress to the next round. If you fail to match all the cards within the assigned timeframe the game will end.


Game Controls

The game can be paused and resumed by clicking the pause/play icons on the left. The game can also be restarted by clicking the restart icon on the right. You can return to the entry page by clicking on the home icon.
