How to Play

In this game there are 8 cards, each card has a match that you as the player must find. All of matches must be found before the timer runs out in order to progress to the next round. The player can receive bonus points if the matches are found quickly. If the player fails to match all the cards within the assigned timeframe the game will end.


Game Controls

The player can pause and resume the game by clicking the pause/play icons on the left. The player can also restart the game by clicking the restart icon on the right.
