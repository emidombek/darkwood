## Description of the Game 

Darkwood is Slavic Mythology inspired card matching game. It has 8 cards layed out with 4 matching pairs that need to be found within a specific timeframe in order for the player to progress into the next round. The game has a simple points system with 1 point being awarded for each matching pair.
